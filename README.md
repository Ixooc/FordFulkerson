El proyecto busca conectar la teoría del flujo máximo con su aplicación práctica mediante una herramienta interactiva. 

La aplicación permite:
- Crear un grafo dirigido de forma manual o aleatoria (8 a 16 nodos).
- Agregar o eliminar aristas y definir sus capacidades.
- Seleccionar un nodo fuente y un nodo sumidero.
- Verificar la conectividad del grafo.
- Ejecutar el algoritmo de Ford-Fulkerson mostrando:
  - Caminos aumentantes
  - Grafo residual
  - Valor final del flujo máximo
  - Corte mínimo asociado
